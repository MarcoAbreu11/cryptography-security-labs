1. INTRODUÇÃO

Este é um exemplo de texto estruturado para ser guardado num 
ficheiro com a extensão .txt. O formato de texto simples é 
ideal para garantir a compatibilidade entre diferentes 
sistemas operativos e dispositivos.

4. PARÁGRAFO DE TESTE (LONG)

A persistência da memória digital permite que informações 
sejam transmitidas através de redes complexas em milissegundos. 
Este ficheiro serve como um contentor minimalista para dados 
que não requerem a sobrecarga de processadores de texto pesados. 
Simplicidade é a sofisticação máxima.
